# Intelligent Sensing Lab Website

A fully responsive, multi‑page website developed as the final project for a university web development course (CS2204). The site showcases research divisions, allows internship applications with rank selection, and provides visitor information.

**Technologies used:** HTML5, CSS3, JavaScript (ES6)  
**Features:** Tabbed interface, dynamic table updates, form validation, rotating promotions, video carousel, print‑friendly styles.

🔗 [Live Demo](https://AmmarInamullah.github.io/intelligent-sensing-lab/)
